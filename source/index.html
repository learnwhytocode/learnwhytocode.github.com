This primer to practical programming was originally designed for an Online News Association class that I’ll be teaching on Sept. 20. If you're planning to attend ONA in San Francisco and are interested in this class, sign up here.

This primer is intended for absolute beginners. The few libraries and programs used in it can be run on any major operating system.

Ruby is the programming language used, but the concepts here apply the same to any other high-level programming language.
